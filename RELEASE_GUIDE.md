Fixbot Release Guide (GitHub Releases)

1. Upload the ZIP

- Create a new GitHub Release.
- Upload: fixbot-installer.zip

2. Share the download link

- Use the release asset URL from GitHub.

3. Friend install steps

- Download fixbot-installer.zip
- Extract it
- Open installer/index.html and click Install Fixbot
  (or run installer\install.cmd from an elevated CMD)

Notes

- The installer will copy Fixbot to Program Files, create an Admin shortcut,
  and add a fixbot command to PATH for admin CMD usage.
